# Bulwk Trading Agent (Web GUI)

Beautiful visual interface for autonomous DeFi trading with local execution.

## Quick Start

**macOS/Linux:**
1. Double-click `Start Bulwk Agent.command`
2. Your browser opens automatically
3. Follow the setup wizard

**Windows:**
1. Double-click `Start Bulwk Agent.bat`
2. Your browser opens automatically
3. Follow the setup wizard

## How It Works

1. **Local Web Server** - Runs on your computer (localhost:3420)
2. **Visual Interface** - Full React GUI in your browser
3. **Secure Execution** - Private keys never leave your machine
4. **Platform Integration** - Links to https://app.bulwk.com

- Node.js 18+ ([Download](https://nodejs.org))
- Modern web browser (Chrome, Firefox, Safari)

## Troubleshooting

**Port Already in Use:**
- Another app is using port 3420
- Close other apps or change PORT in server/index.js

**Node.js Not Found:**
- Install from https://nodejs.org
- Restart terminal/command prompt

**Dependencies Failed:**
- Run: `npm install`
- Check internet connection
